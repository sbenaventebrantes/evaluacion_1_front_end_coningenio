# ConIngenio - Evaluación Front-End

ConIngenio es una landing page creada con HTML, CSS y JavaScript como parte de la evaluación de la unidad 1. El objetivo fue construir una experiencia clara, funcional y adaptable, con una interfaz sencilla de recorrer y pensada para mostrar los servicios y la información principal de la empresa.

## Descripción
El sitio presenta una estructura institucional con secciones de bienvenida, servicios, información corporativa y un formulario de contacto. La propuesta busca comunicar de forma directa lo que hace la empresa, sin perder una presentación visual ordenada y accesible.

## Secciones
- Home
- Nuestros Servicios
- Nosotros
- Contáctenos
- Footer

## Funcionalidades
- Menú responsive para facilitar la navegación en distintos dispositivos
- Cambio entre tema claro y oscuro
- Tarjetas de servicios con detalle expandible
- Formulario de contacto con validaciones básicas
- Contador de caracteres en el campo de mensaje
- Registro de los datos enviados en la consola del navegador
- Resaltado del enlace activo según la sección visible
- Navegación por anclas entre las secciones principales

## Estructura principal
- `index.html`
- `css/style.css`
- `js/main.js`
- `assets/logo.svg` (referenciado en el HTML)

## Cómo ejecutar
1. Abrir `index.html` en un navegador.
2. Revisar la navegación, el comportamiento responsive, el cambio de tema y el formulario de contacto.
