# evaluacion_1_front_end_coningenio
Proyecto realizado para la evaluación de Front-End, donde se desarrolla una página web aplicando HTML, CSS y JavaScript, incorporando interactividad, validaciones y diseño adaptable a distintos dispositivos.

## Estructura base
- `index.html`: estructura principal de la página.
- `styles.css`: estilos base y diseño responsivo simple.
- `script.js`: interacción básica con el botón principal.
