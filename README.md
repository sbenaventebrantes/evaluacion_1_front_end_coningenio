# ConIngenio - Evaluación Front-End

Proyecto desarrollado con HTML, CSS y JavaScript para la evaluación de la unidad 1.

## Contenido del sitio
- Home
- Nuestros Servicios
- Nosotros
- Contáctenos
- Footer

## Funcionalidades
- Menú responsive
- Tema claro / oscuro
- Formulario con validaciones
- Registro de datos del formulario en consola
- Secciones de servicios con detalle expandible
- Contador de caracteres en el mensaje
- Resaltado del enlace activo al hacer scroll

## Archivos principales
- `index.html`
- `css/style.css`
- `js/main.js`

## Cómo ejecutar
1. Abrir `index.html` en un navegador.
2. Probar navegación, responsive, cambio de tema y formulario.

## Observación
Las evidencias complementarias (mockup, pruebas UX y análisis de mejoras) deben adjuntarse aparte según el formato solicitado por la evaluación.
